バイナリ法

バイナリ法

$x=a^k$のとき，2乗計算を$k$回行うことになる． 計算を効率よくする方法として，$a^{2^i}$を順次求めることで，計算量を$log(k)$回に抑える方法がバイナリ法である．

具体例

$5^{21}=5^{2^4}*5^{2^2}*5^{2^0}$

2進数に展開し，左から順に展開することにより計算を実行する． これにより，$g^k(mod p)$を計算する．

アルゴリズム

def Binary(k, g, p):
    k_binary = []
    while(k != 0):
        k_binary.append(k%2)
        k = k//2
        if k == 1:
            k_binary.append(k)
            k = 0
    y = 1
    for i in reversed(range(len(k_binary))):
        if k_binary[i] == 1:
            y = (y*y%p)*g%p
        else:
            y = (y*y%p)
    return y
